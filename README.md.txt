# 🏡 Premier Properties - Sitio Web Inmobiliario con Recorridos 3D

![Sitio Web Inmobiliario](https://placehold.co/800x400/4F46E5/FFFFFF?text=Premier+Properties+-+Recorridos+3D)

Un sitio web inmobiliario moderno y responsive con recorridos virtuales 3D interactivos que permiten a los clientes explorar propiedades como si estuvieran allí en persona.

## 🌟 Características Principales

- ✅ **Recorridos 3D Simulados**: Navegación interactiva por las propiedades
- 🖱️ **Interacción por Mouse**: Haz clic y arrastra para explorar cada habitación
- 📱 **Diseño Responsive**: Funciona perfectamente en móviles, tablets y desktop
- 🎨 **Interfaz Moderna**: Diseño atractivo con animaciones y efectos visuales
- 🚀 **GitHub Pages**: Listo para publicar con una URL real

## 🚀 Cómo Usar

### Explorar Propiedades
1. Navega por las propiedades destacadas en la página principal
2. Haz clic en "Ver Recorrido 3D" en cualquier propiedad
3. Explora la propiedad haciendo clic y arrastrando para mirar alrededor
4. Usa las flechas o puntos para moverte entre habitaciones

### Controles del Recorrido 3D
- **🖱️ Click + Arrastrar**: Mira en diferentes direcciones
- **⬅️➡️ Flechas**: Cambia entre habitaciones
- **●●● Puntos**: Salta a habitaciones específicas

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura del sitio
- **Tailwind CSS** - Framework de estilos
- **JavaScript Vanilla** - Interactividad y recorridos 3D
- **Font Awesome** - Íconos
- **GitHub Pages** - Hosting gratuito
